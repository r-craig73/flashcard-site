JavaScript (JS)

is an object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operators

    Assignment Operators

    =

    assigns variable's value to the left based on the value to the right.

    +=

    increases variable's value to left based on the value to the right.

    -=

    decreases variable's value to left based on the value to the right.

    *=

    multiplies variable's value to the left based on the value to the right.

    /=

    divides variable's value to the left based on the value to the right.
